说明

http://192.168.3.33:3000/regist

方法

参数传递方法

url

参数

名称

类型

说明

nick

string

昵称

email_phone

string

邮箱/手机号码

password

string

密码

string

头像，支持.png和.jpg格式

validate

string

验证码

返回值格式

JSON

返回值

名称

类型

说明

errcode

integer

错误码，成功时errcode=1,用户已注册errcode=0,邮箱或手机号格式不正确errcode=-1

essmsg

string

错误信息

http://192.168.3.33:3000/regist?email_phone=1130200106@qq.com&password=wu201314&validate=123456

{"errcode":-1,"errmsg":"验证码错误"}
